# video-rental

Demo application for video-rental 

## Usage

### Run the application locally

`lein ring server`

### Run the tests

`lein test`

### Packaging and running as standalone jar

```
lein do clean, ring uberjar
java -jar target/server.jar
```

### Packaging as war

`lein ring uberwar`

### Deploy to Heroku

```sh
$ heroku create
$ git push heroku master
$ heroku open
```

or
